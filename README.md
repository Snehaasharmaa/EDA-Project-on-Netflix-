# Netflix Data Analysis Project

## Objective
Analyze Netflix content to understand trends in genres, duration, ratings, and release years.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Movies dominate Netflix content
- Drama and Comedy are the most popular genres
- Most content duration ranges between 80–120 minutes

## Dataset
Netflix Movies and TV Shows dataset
